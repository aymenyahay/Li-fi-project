# Li-fi-project
Bu projede Li-Fi (Light Fidelity) teknolojisini temsilen, ışık üzerinden veri iletim mantığı Arduino kullanılarak gösterilmiştir. LDR sensörü ile algılanan ışık verileri Arduino tarafından işlenmiş ve LCD ekran üzerinde görüntülenmiştir.
